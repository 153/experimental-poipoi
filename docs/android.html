Kiwi Browserを使う方法

  1. Kiwi Browserをインストール
  2. Tampermonkeyをインストールしてた方はアンインストール (Chromeから削除を押す)
  3. Kiwi BrowserでViolentmonkeyをインストール
  4. Kiwi Browserを閉じて(タスクキル)開きなおす
  5. experimental-poipoiをクリックしてインストール

Firefox Nightlyを使う方法

  1. Firefox Nightlyをインストール
  2. Firefox NightlyでTampermonkeyをインストール
  3. Firefox Nightlyを閉じて(タスクキル)開きなおす
  4. experimental-poipoiをクリックしてインストール
